## Alibaba Cloud microgw SDK for Python

## 安装

- **使用 pip 安装(推荐)**

如未安装 `pip`, 请先至pip官网 [pip user guide](https://pip.pypa.io/en/stable/installing/ "pip User Guide") 安装pip .

```bash
# 安装 alibabacloud_microgw20200810
pip install alibabacloud_microgw20200810
```
